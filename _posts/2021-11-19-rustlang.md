## cargo watch

Cargo Watch watches over your project's source for changes, and runs Cargo commands when they occur.

**To install**

'''rust
cargo install cargo-watch
'''

**To run program and watch of source code changes**

'''rust
cargo watch -q -c -w src/ -x run
'''
